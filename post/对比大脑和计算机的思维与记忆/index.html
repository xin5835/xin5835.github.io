对比大脑和计算机的思维与记忆

February 17, 2020

对比大脑和计算机的思维与记忆，提升自己学习过程中的思维水平和记忆能力。

大脑具有思维、记忆（包括瞬时记忆和长期记忆）和电脑的cpu和储存（ram和硬盘）。

学习的过程 = 理解 + 记忆

计算机 = cpu + ram + ssd

精益一件事 = 渴望 + 持续的重复

大脑负责创造和构思，不负责记忆，就像cpu只负责计算，不负责储存一样，储存交由内存和硬盘来负责，缓解cpu的运行压力，记忆（机械的重复过程），可减轻大脑运行的压力，防止每天有很多事情想，心累、心塞的感觉，否则所有的事情交由大脑/cpu来处理，大脑和cpu会逐渐升温，直至瘫痪。

学习的过程需要将理解和记忆间歇性的分开，至少在学习的过程中目的不是为了记住它，因为记忆不是大脑的功能。

不渴望、无法触动自己的神经、不能产生共鸣，则无法激活自己的神经元、脑细胞，大脑则不运转，犹如无水流动的河流，无电传播的电网，不能产生能量，水是死水，会发臭；电网无用，不能使电器工作。

小溪不断流淌，拓宽水道，终将汇聚成宽广的河流，碎片的记忆不断重复、持续，终将点连成线、成面，成系统，在这个过程就像小溪不追求秒流入江河大海一样，也无需抱有在开始记忆以及之前的学习步骤中一下记住系统的知识一样，早晚会达成这样的目的，抱着这样的目的，将会严重的影响自己消化吸收。——-真这样就是昙花一现。
